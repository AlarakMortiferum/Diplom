# Кандидаты в отдельные bug-oriented UI-тесты

## BUG-001
Из экрана `News` пункт `About` в popup-меню не открывает экран About.

Причина найдена в коде `NewsListFragment.kt`: отключается `mainMenu.menu.getItem(2)`, а третий элемент меню — это `About`, не `News`.

## BUG-002
В форме создания/редактирования новости поле `Description` по ручной проверке не принимает ввод.

Для автотестов это можно вынести в отдельный класс `KnownBugsTest` после того, как стабильно заработает основной набор.
